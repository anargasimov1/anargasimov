Proyektər proyektlərim haqqında qısa məlumat vermək itəyirəm

Doshabcetring

Doshabcetring, şirkətlərə və fərdi şəxslərə yemək hazırlayıb çatdırmaqla məşğul olan bir şirkətdir. Müştərilərə keyfiyyətli və dadlı yeməklər təklif edən bu şirkət üçün veb saytını mən hazırlamışam. Bu layihədə mənim rolum aşağıdakıları əhatə edirdi: Proje Planlaşdırılması və Dizaynı: Müştərinin tələblərini nəzərə alaraq, saytın dizaynını və funksionallığını planlaşdırdım. Back-End İnkişafı: Java və Spring Boot istifadə edərək, saytın arxa hissəsini kodladım, məlumat bazasını və API-ləri inteqrasiya etdim. Front-End İnkişafı: HTML, CSS və JavaScript ilə saytın istifadəçi interfeysini hazırladım, mobil cihazlara uyğunlaşdırdım. Test və Təkmilləşdirmə: Saytın tam işlək olmasını təmin etmək üçün hərtərəfli testlər keçirdim, potensial səhvləri aradan qaldırdım və istifadəçi təcrübəsini optimallaşdırdım.

Compo

compo.ru texniki dəstək və konsultasiya xidmətləri də göstərir, müştərilərinə doğru məhsul seçimi və istifadəsi ilə bağlı yardım edir. Şirkət onlayn mağaza kimi fəaliyyət göstərərək, istifadəçilərə rahat və sürətli alış-veriş imkanı təqdim edir. Və həmin it kompaniyanın mənə online mağaza olan saytın bəzi səhifələrini front hissəsini məhz mənə həvalə etmişdir. Və məndə bu tapşırığın öhdəsində uğurla gəlmişdimş
